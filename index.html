Portable Power Bank

This project involves designing a power bank with a Li-ion battery, discrete boost converter, and STM32 feedback loop to provide stable voltage output. The project is used for educational purposes, teaching students about power systems and embedded controllers.

Blue Sky Solar Racing

As part of the Blue Sky Solar Racing team, I designed the VFM Motor Controller PCB and assisted with dynamometer testing. I also contributed to mechanical integration and worked on various control algorithms for MPPT systems.

SDR Radio

This project involved working on a Software-Defined Radio (SDR) system. Initially experimenting with AC switches, I transitioned to using a pre-made RF switch for performance enhancement. The project highlights my exploration into RF systems and circuit optimization.

FPGA Maze Solver Game

This project utilized an FPGA to design a maze solver game using Verilog. The game implemented algorithms that allowed the maze to be dynamically solved, demonstrating my skills in digital design and FPGA programming.
